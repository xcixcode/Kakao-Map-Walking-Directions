# Kakao-Map-Walking-Directions

**카카오맵 기반 동의대 도보 길찾기**

동의대학교 주요 건물 간 도보 경로를 손쉽게 찾을 수 있는 HTML, CSS, JavaScript 기반의 웹 프로젝트입니다.  
출발지와 도착지를 선택하면 카카오맵이 새 창으로 열려 두 지점 간 도보 경로를 안내합니다.

## 프로젝트 개요

**Kakao-Map-Walking-Directions**는 동의대학교 캠퍼스 내 주요 건물 간의 도보 경로를 간편하게 확인할 수 있도록 제작된 웹 애플리케이션입니다.  
사용자가 출발지와 도착지를 선택하면, 카카오맵의 도보 길찾기 페이지로 자동 연결되어 해당 경로를 바로 확인할 수 있습니다.

본 프로젝트는 **카카오맵의 웹 링크 기능을 활용**하여 작동하며, **카카오 지도 API를 직접 사용하지 않습니다.**  
별도의 서버 환경 없이 클라이언트 사이드에서 작동하므로, 단순하고 빠르게 실행할 수 있습니다.  
인터넷 연결이 필요하며, **PC 환경에 최적화된 UI**로 설계되었습니다.

---

## 기술 스택
- **HTML5** - 페이지 구조 및 콘텐츠 구성
- **CSS3** - 스타일링 및 레이아웃 디자인
- **JavaScript (Vanilla)** - UI 상호작용 및 경로 생성 로직

---

## 주요 기능
- 동의대학교 **캠퍼스 내 주요 건물 위치 지정 및 선택 기능**
- 출발지와 도착지를 선택할 수 있는 **드롭다운 메뉴 제공**
- 선택된 위치를 기반으로 **카카오맵 도보 경로 자동 연결**
- 출발지와 도착지가 동일할 경우 **경고 메시지 표시**
- **간단하고 직관적인 UI 디자인**

## 설치 및 실행

### GitHub Pages  
https://xcixcode.github.io/Kakao-Map-Walking-Directions/

### 로컬에서 실행
1. 저장소를 클론하거나 파일을 다운로드합니다.
2. `index.html` 파일을 웹 브라우저에서 엽니다.
3. 출발지와 도착지를 선택합니다.
4. **"경로 찾기"** 버튼을 클릭하면 새 창에서 카카오맵이 열립니다.
5. 두 지점 간 도보 경로를 확인합니다.
